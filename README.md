# ترجمه ماینکرفت به زبان فارسی

این پروژه برای ترجمه ماینکرفت نسخه ۱.۸.۹ به زبان فارسی می‌باشد.

اگر داوطلب این کار هستید فایل [fa_IR.lang](/fa_IR.lang) را با ویرایشگر متنی دلخواهتان ویرایش کنید و به این مخزن پول ریکوئست بزنید.

قوانین:

1.برای ترجمه به هیچ عنوان از کلمات مصوبه فرهنگستان فارسی استفاده نکنید.

2.کلمات خاصی که توسط تیم موجانگ درست شده اند، مانند **Ender peral** یا **Piglin Brute** نباید ترجمه شوند اما کلماتی مانند **Zombie** یا **Water Bucket** خاص نیستند و قابل ترجمه هستند.
